# Beginner-python-code.
Simple Password Generator.

This project is a simple Python script that creates strong, random passwords. It’s designed for beginners and anyone who wants a quick way to generate secure passwords without installing extra tools.
What it does
- Creates a random password every time you run it
- Uses letters, numbers, and symbols
- Lets you change the password length if you want
- Runs on any computer with Python installed
How to use it
- Make sure Python 3 is installed on your system
- Run the script in your terminal:
- The program will instantly show you a new password
